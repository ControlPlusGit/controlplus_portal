u32  baudrate,
u32 *  actbaudrate 
)

Initialize UART interface for Tx and Rx.

The function calls uartTxInitialize() and uartRxInitialize() - the second one only if uart rx functionality is enabled.

Parameters
[in]sysclk,:clk frequency the system is configured to
[in]baudrate,:Baudrate to set, e.g. 115200
[out]actbaudrate,:Actual set baudrate which normally differs from requested baudrate due to divider error.
Returns
ERR_NONE : No error, UART initialized.

s8 uartTxInitialize ( u32  sysclk,
u32  baudrate,
u32 *  actbaudrate 
)

Initialize UART TX interface.

This function initializes the UART interface, i.e. sets the requested baudrate and enables transmission only.

Note
System frequency must not be changed since baudrate calculation is done on base of a fixed frequency.
Parameters
[in]sysclk,:clk frequency the system is configured to
[in]baudrate,:Baudrate to set, e.g. 115200
[out]actbaudrate,:Actual set baudrate which normally differs from requested baudrate due to divider error.
Returns
ERR_NONE : No error, UART initialized.
