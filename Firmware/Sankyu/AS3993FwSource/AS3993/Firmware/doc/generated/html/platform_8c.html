Detailed Description

Implementation of serial interface communication with AS3993.

The file provides functions to initialize the uController and the board for SPI communication with AS3993. The function writeReadAS3993() provides communication functionality. The function writeReadAS3993Isr() reads interrupt status register and is only called from the ISR. (no reentrant function necessary).
For porting to another uController/board the functions in this file have to be adapted.
