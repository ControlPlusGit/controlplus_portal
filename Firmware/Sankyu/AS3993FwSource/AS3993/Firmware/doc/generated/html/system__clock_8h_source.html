58  #define POSC_NOMINAL xxxxULL
59  where xxxs is the frequency in hertz of the oscillator.
60  Also you must configure the divider in the Config Word2 to the
61  appropriate value so that the PLL block has as input frequency 4 MHz.
62  E.g. if your oscillator has 16MHz output you must use PLLDIV_4
